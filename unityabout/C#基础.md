# 一、C#语言及其特点

C#是微软公司发布的一种面向对象的、运行于.NET Framework和.NET Core（完全开源，跨平台）之

上的高级程序设计语言

C#是一种安全的、稳定的、简单的、优雅的，由C和C++衍生出来的面向对象的编程语言。它在继承C 

和C++强大功能的同时去掉了一些它们的复杂特性

C#是面向对象的编程语言
