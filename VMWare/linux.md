#  命令累积

## 1. 开启/关闭图形化界面

```
1.关闭图形化界面  systemctl set-default multi-user.target
2.开启图形化界面  systemctl set-default graphical.target
```

##  2. 查看文件和目录

```
查看当前所在的路径：pwd
命令格式：命令 -选项 -参数（目录）
查看当前路径下的文件：ls
-d：只看当前目录的信息
-l：看详细信息
-a：显示所有 任何一个文件前面加上“.”，表示隐藏文件
-h：显示文件大小
```
